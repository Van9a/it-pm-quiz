# 🎓 PUET AI Expert 2026 — Adaptive NMT Trainer

![Version](https://img.shields.io/badge/version-3.0-blueviolet)
![Tech](https://img.shields.io/badge/tech-Gemini_AI-orange)
![Status](https://img.shields.io/badge/status-active-success)

**PUET AI Expert** — це інтелектуальна освітня платформа, розроблена для підготовки абітурієнтів до НМТ 2026 та навчання основам спеціальності **073 "Менеджмент"** у ПУЕТ. Проєкт поєднує в собі систему тестування (Quiz) та повноцінну базу знань (LMS), що працює на базі нейромережі **Gemini 3 Flash**.

## 🚀 Основні можливості

- **🧠 Динамічна генерація питань:** Жодних статичних тестів. AI щоразу генерує унікальні питання рівня НМТ з обраної теми.
- **📚 Інтерактивна Академія:** Можливість отримати розгорнуту лекцію, приклади та бізнес-кейси по будь-якій темі з плану підготовки.
- **📐 MathJax Integration:** Повна підтримка складних математичних формул і графіків прямо в браузері.
- **📊 AI-Аналітик:** Після кожного тесту нейромережа аналізує твої результати та дає персональні поради для вступу.
- **🏗 Модульна архітектура:** Код розділений на незалежні шари (HTML/CSS/JS) та хмарний бекенд на Google Apps Script.

## 🛠 Технологічний стек

* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).
* **Backend:** Google Apps Script (GAS).
* **AI Engine:** Gemini AI API (Google Cloud).
* **Math Engine:** MathJax 3.0.
* **Hosting:** GitHub Pages.

## 📂 Структура проєкту

* `index.html` — інтерфейс та структура додатку.
* `style.css` — сучасний адаптивний дизайн.
* `script.js` — логіка взаємодії з API, обробка даних та керування станом.
* `Код.gs` (Back-end) — серверна частина для безпечних запитів до AI.

## ⚙️ Як це працює

Проєкт використовує архітектуру "Клієнт — Хмарний посередник — AI":
1. Клієнт (JS) надсилає запит на **Google Apps Script**.
2. GAS безпечно дістає API-ключ із налаштувань проекту та звертається до **Gemini AI**.
3. Отримана відповідь форматується та відображається користувачеві з використанням рендерингу MathJax.
